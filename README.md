# OldTech Medical Catalog — Apple-style Light Design

Static bilingual medical equipment catalog redesigned in a clean light product-page aesthetic.

## Files

- `index.html` — page structure, sticky navigation, medical sections and request form
- `styles.css` — light fog canvas, white 28px cards, large display typography, blue primary CTAs
- `app.js` — RU/EN switch, product filters, request cart, card counters and form validation
- `assets/favicon.svg` — simple medical brand mark

## Deploy

Upload the folder contents to GitHub and deploy with Vercel as a static site. No build step is required.

## Notes

The request form is static. Connect it to email, CRM, or a backend endpoint when moving to production. The personal-data consent checkbox is required before form submission.
